# Studai App

## Project Overview

Studai App is a comprehensive application designed to manage student data, including profiles, performance tracking, and communication tools. The goal of this application is to provide a seamless experience for students, teachers, and administrators.

## Features
- User authentication
- Profile management
- Performance analysis
- Communication tools

## Getting Started

To get started with this project, follow the instructions below for installation and usage. 

## Installation

1. Clone the repository
   ```bash
   git clone https://github.com/cpht18/studai-app.git
   ```
2. Navigate to the project's directory
   ```bash
   cd studai-app
   ```
3. Install dependencies
   ```bash
   npm install
   ```
4. Run the application
   ```bash
   npm start
   ```
